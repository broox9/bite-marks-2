# Fixes

Use this folder for bug-fix notes, support issue writeups, and ticket-resolution docs.

## Filename Format

Use `YYYY-MM-DD-kebab-case.md`.

Examples:

- `2026-06-07-login-session-timeout.md`
- `2026-06-07-abc-123-login-session-timeout.md`

Include the ticket or issue identifier in the slug when one exists.
