# User Interface

## List page UI/UX
create a list screen for a restaurant bookmark app that displays a users saved restaurant listings, with a search bar that slides up, shows results as they stream in, and has an overlay with details on any selected restaurant with "save" and "clear" buttons


## Details page UI/UX
create a screen for a restaurant bookmark app that lists a saved restaurant's information and displays or allows input of a users rating, notes, visited (or not) status, and social media links
